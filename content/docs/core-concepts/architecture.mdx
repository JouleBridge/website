# Architecture

JouleBridge follows a four-surface split:

1. Asset Agent
2. Site Router
3. Portfolio Orchestrator
4. Console

## Asset Agent

Lives closest to the equipment and protocol interfaces.

Its job is to:

- read site and asset state
- normalize mixed-vendor inputs
- participate in the local proof path

## Site Router

The site coordination surface.

Its job is to:

- apply policy and constraints
- turn intent into feasible actions
- preserve the local trust boundary

## Portfolio Orchestrator

The cloud coordination surface.

Its job is to:

- distribute policy
- ingest proofs
- send portfolio guidance back to sites

## Console

The operator surface.

Its job is to:

- expose site and portfolio state
- help support teams understand what happened
- make proofs and incidents reviewable

## Design rule

Local safety beats remote intent. The cloud is useful because the site runtime is strong first.
