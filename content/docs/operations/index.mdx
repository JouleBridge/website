# Operations

Running Bridge Kernel in production requires proper deployment, monitoring, security hardening, and operational procedures. This section covers each area.

## Guides

- **[Deployment](/docs/operations/deployment)** — Deploy on edge gateways (ARM/x86), on-premise servers, cloud instances, Docker, or Kubernetes.
- **[Observability](/docs/operations/observability)** — Prometheus-compatible metrics, structured event logs, health degradation signals, and SLO-linked alerting.
- **[Security](/docs/operations/security)** — Ed25519 key management, key rotation, replay protection, mutual TLS, and TPM/HSE hardware signing.
- **[Troubleshooting](/docs/operations/troubleshooting)** — Common issues, diagnostic commands, and resolution steps.
