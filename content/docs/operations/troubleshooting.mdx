# Troubleshooting

When a deployment fails, start with three questions:

1. Did the site runtime observe the expected state?
2. Did local coordination reject or accept the action?
3. Did the proof path seal and export the resulting record?

Troubleshooting should preserve evidence instead of wiping it away.
