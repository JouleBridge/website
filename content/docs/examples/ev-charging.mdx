# EV Fleet Charging

This is the strongest current application path for JouleBridge.

## Why this wedge matters

Fleet depots have:

- hard departure windows
- mixed-vendor charging equipment
- optional ESS and solar
- strong pressure to keep coordination local

## What the runtime does

At a depot, JouleBridge should be able to:

1. understand charger and site state
2. coordinate local actions safely
3. record dispatch, acknowledgement, and measured result

## What success looks like

- local coordination continues when connectivity degrades
- proof records explain what happened after the charging window
- the deployment can be repeated on the next depot without rethinking the whole stack
