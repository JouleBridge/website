# P2P Sync

Proofs often need to move without moving the whole trust boundary.

## What sync is for

- exporting local records upward
- reconciling proof history across surfaces
- preserving site autonomy while still supporting portfolio review

## What sync is not for

It is not permission to treat the cloud as the only source of truth.

The site should still remain understandable and supportable on its own.
