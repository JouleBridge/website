# Getting Started

Bridge Kernel is a deterministic, tamper-evident event ledger runtime for energy settlement. These guides take you from zero to a running node with signed, policy-gated events in your local ledger.

## What you'll need

- **Rust toolchain** — `rustup` and `cargo` (stable channel)
- **Git** — to clone the Bridge Kernel repository
- **~10 minutes** — for build, configure, and first run

## Guides

- **[Quick Start](/docs/get-started/quickstart)** — Clone, build, and run your first signed ledger event in minutes.
- **[Beginner Guide](/docs/get-started/beginner-guide)** — A slower walkthrough that explains each step and what's happening under the hood.

## Next steps

Once you have a running node, explore the [Core Concepts](/docs/core-concepts) to understand the architecture, proof system, and policy engine that power Bridge Kernel.
