# FAQ

### What is JouleBridge?

JouleBridge is the runtime, proof, and coordination layer for distributed energy.

### What is DepotOS?

DepotOS is the first commercial wedge built on the JouleBridge platform for EV fleet depots.

### Does the cloud control everything?

No. The platform is explicitly local-first. The cloud layer coordinates and reviews; the site runtime still has to make safe decisions locally.

### Is JouleBridge a hardware company?

No. The current motion is partner-hardware deployment first.

### What is the first commercial use case?

EV fleet depots.
