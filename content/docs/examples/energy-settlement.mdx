# Commercial Site Coordination

This example is an emerging application path, not the primary live wedge.

## Where it fits

Commercial and industrial sites often combine:

- storage
- solar
- metering
- flexible loads
- local operating constraints

## What JouleBridge adds

The same local coordination and proof model used at depots can be applied here when the site needs:

- one runtime across mixed vendors
- local decision-making
- proof-backed operating history

The commercial posture for this path should stay honest: promising, but secondary to the depot wedge.
