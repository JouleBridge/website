# Glossary

**Bridge Kernel** - the edge runtime closest to the equipment. It reads mixed-vendor assets, coordinates local actions, and signs the operating record.

**JouleBridge Cloud** - the cloud service surface for policy distribution, proof ingest, and multi-site visibility.

**JouleBridge Console** - the operator and reviewer surface for site state, incidents, evidence packs, and proof exports.

**Proof Chain** - the signed record path that links intent, execution, and measured outcome.

**EV Depot Wedge** - one initial application path for JouleBridge, alongside industrial-site deployments.
