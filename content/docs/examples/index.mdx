# Examples

Step-by-step walkthroughs that show Bridge Kernel solving real energy settlement problems. Each example includes configuration, adapter setup, policy rules, and expected outputs.

## Walkthroughs

- **[C&I Energy Settlement](/docs/examples/energy-settlement)** — Verify multi-source power contracts for open-access C&I buyers. Covers Modbus adapter setup, billing-period proof generation, and reconciliation evidence packs.
- **[EV Fleet Charging](/docs/examples/ev-charging)** — Meter, sign, and settle high-frequency charge sessions for depot and fleet operators. Uses OCPP adapter with per-session proof envelopes.
- **[Grid Event Audit](/docs/examples/grid-audit)** — Reconstruct the precise state of grid-connected assets at any point in time. Demonstrates signed event chains for regulatory audit trails.
