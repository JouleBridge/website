# How the JouleBridge proof chain uses Ed25519

The proof path needs signatures that are fast, compact, and widely understood.

Ed25519 is useful because it helps the runtime do three things well:

- sign frequently
- verify clearly
- keep the resulting proof record lightweight

The bigger point is not the algorithm by itself. The bigger point is that the site should be able to leave behind a record another party can inspect later.
